## О проекте

Данный репозиторий содержит материалы дипломной работы, посвящённой применению **графовых нейронных сетей** для прогнозирования управляющих параметров ветроэнергетических установок (углов установки лопастей).

**Основная идея:** учитывать не только временную динамику, но и **пространственные взаимодействия** между турбинами (эффект ветровой тени — wake effect) с помощью модели **GraphSAGE**.


## Основные результаты

- Использован датасет **SDWPF** (134 турбины, 245 дней)
- Построен пространственный граф (k-NN, k=6)
- Реализована модель **GraphSAGE** (3 слоя, hidden=128)
- Получены метрики: **MAE = 15.76**, **RMSE = 28.25**, **R² = 0.6**
- Разработано интерактивное веб-приложение на **Streamlit**
